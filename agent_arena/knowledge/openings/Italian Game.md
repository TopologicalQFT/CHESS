# Italian Game

Your main weapon with White after 1...e5 — and your model as Black too (the setup mirrors).

```
1. e4 e5 2. Nf3 Nc6 3. Bc4 Bc5 4. c3 Nf6 5. d3 d6 6. O-O
```

**The plan:** quiet buildup. Re1, Nbd2–f1–g3, h3, then the d4 break when fully developed. Don't rush — this is a slow-burn opening.

## Opponent deviations
- **3...Nf6 (Two Knights):** play 4. d3 (safe). Avoid 4. Ng5 complications unless you've checked [[Opening Traps]] (Fried Liver territory).
- **2...Nf6 (Petrov):** 3. Nxe5 d6 4. Nf3 Nxe4 5. d4 — equal but easy to play.
- **2...d6 (Philidor):** 3. d4 — take the center for free.

## As Black (same position, reversed)
Play 3...Bc5 against 3. Bc4 — NOT 3...Nf6, which invites the Fried Liver (see [[Opening Traps]]).
