## Every-move checklist
1. Can any pawn (mine or theirs) promote by force? Count the race exactly — [[The Square Rule]].
2. Is my king as active as it can be? Any stalemate risk (mine or theirs)?
3. Before trading the last pieces: is the resulting pawn ending actually won/drawn? Calculate with [[Opposition]], don't hope.
4. Which pawn majority can create a passer? Start it early ([[Endgame Golden Rules]]).
